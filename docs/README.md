# ChIC Documentation

Welcome to the documentation for the **Charité Imaging Classification (ChIC)** — an interactive web application for the prognostic assessment of Polycystic Liver Disease (PLD) progression.

> 🔗 **Live app:** <https://halbritter-lab.github.io/ChIC/> · **Project home:** [repository README](../README.md)

ChIC is an **informational, educational, and research tool** — it is not a diagnostic device. Please read the [Disclaimer](disclaimer.md) before use.
